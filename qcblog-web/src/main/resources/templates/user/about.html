博苑平台介绍

        如今，网络社交已经成为现代信息传播的一种新形式、新媒介，与传统的社交方式不同，网络传播的途径多种多样，例如点对点、面对点、双向甚至多向的传播途径成为网络中重要的几种特征，博客便是其中表现形式之一。微博客的信息获取、舆论、公益活动等渠道是现在人们越来越关注的内容。博客平台作为本课题的重点研究内容，并与学校所学的软件工程、计算机技术、信息系统等课程的理论知识以及工作实习实践经验相结合，对其进行扩展开发和技术研究。

        其主要完成的工作与实现的功能如下：
    （1）使用SpringBoot主流框架完成平台后端开发环境的快速搭建，后续结合AngularJs框架实现前端与后端数据交互、双向绑定的简单操作。以便于为后面开发各种平台接口做技术支撑。
    （2）后端对已经开发完成的平台接口，测试其对数据库CRUD操作的响应速度。对响应时间久的接口进行调度分析，并对此进行迭代优化，最终降低到最小交互时长，目的是为了有效提高用户的体验度。
    （3）前端则为用户提供丰富且简约的响应式界面和可视化的后台管理系统。主要包括检索技术文章、查阅专题类别文章、发表自写文章、个人主页以及休闲娱乐等功能模块。后台管理则可以对自己的个人信息展示、自写文章、评论等进行增删改操作。
    （4）针对管理员角色也提供了一套简洁、操作方便的可视化响应式页，拥有用户管理、评论管理、文章审核、日志管理、权限管理等模块。
    （5）针对技术文章的持续更新需求，则通过WebMagic框架周期性地从各大正规平台如知乎、CSDN、博客园等等爬取最新技术文章。
    （6）为用户提供个性说说与评论留言两大反馈模块，为平台建设提供意见建议，让平台技术不断循环迭代改进与更新。
